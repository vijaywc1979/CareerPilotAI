# CareerPilot AI

CareerPilot AI is an AI-powered career assistant that helps professionals discover the best job opportunities, score resume compatibility, identify referral opportunities, optimize resumes, prepare applications, and track career progress.

## Features

- AI Resume Intelligence
- Job Discovery
- Resume Matching
- ATS Optimization
- Referral Finder
- Cover Letter Generator
- Interview Coach
- Career Dashboard

## Technology Stack

- Python
- FastAPI
- Streamlit
- SQLite
- LangGraph
- Playwright
- Sentence Transformers
- FAISS
